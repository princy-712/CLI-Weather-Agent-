# Weather AI Agent 🤖

## Project Description
AI powered Weather Agent using Gemini AI with Tool Calling Architecture.

## Features
- Understands user queries
- Uses planning and reasoning steps
- Fetches weather data
- Executes system commands

## Tech Stack
- Python
- Gemini AI
- Requests
- dotenv

## Setup

### Clone Repository
git clone https://github.com/princy-712/CLI-Weather-Agent.git

### Create Virtual Environment
```
python3 -m venv venv
```

### Activate Virtual Environment

Mac/Linux:
```
source venv/bin/activate
```

Windows:
```
venv\Scripts\activate
```

### Install Dependencies
```
pip install -r requirements.txt
```

### Create .env File
```
GEMINI_API_KEY=your_api_key_here
```

### Run Project
python3 app.py

## Important
Do not push API keys or venv folder to GitHub.
